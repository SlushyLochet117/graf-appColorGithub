**1.1 Introducción a la programación de interfaces Web**
La aplicación desarrollada es una página web interactiva que permite al usuario seleccionar y visualizar colores mediante el modelo RGB (Rojo, Verde y Azul). Esta aplicación fue construida utilizando tecnologías web como HTML para la estructura, CSS junto con Bootstrap para el diseño visual y JavaScript para la lógica e interactividad.

El sistema cuenta con distintos métodos de entrada, incluyendo controles deslizantes, campos numéricos y un selector de color nativo de HTML, los cuales se encuentran sincronizados entre sí. A medida que el usuario modifica los valores RGB, la aplicación muestra en tiempo real el color resultante dentro de un área de visualización circular, así como su correspondiente código hexadecimal.

Adicionalmente, la aplicación incorpora funcionalidades avanzadas orientadas a mejorar la experiencia del usuario, como el almacenamiento del último color seleccionado, un historial de colores recientes y la identificación aproximada del nombre del color. Gracias a estas características, la aplicación facilita la selección, comparación y reutilización de colores, demostrando el uso práctico de conceptos fundamentales del desarrollo web moderno.
